# Easy Math

Easy Math is a simple browser-based math quiz game.  
The player needs to decide whether the displayed calculation and result are correct or incorrect before the timer runs out.

## Live Demo

Play here: https://luuminhtri.github.io/math-quiz/

## Features

- Random math calculations
- True / False answer buttons
- Score and level system
- Countdown timer
- Increasing difficulty by level
- Sound effects for correct and wrong answers
- Game over popup with replay option

## How to Play

1. A math expression will appear on the screen.
2. A result will be shown below it.
3. Click:
   - ✅ if the result is correct
   - ❌ if the result is wrong
4. If your answer is correct, you move to the next level.
5. If your answer is wrong or time runs out, the game ends.
